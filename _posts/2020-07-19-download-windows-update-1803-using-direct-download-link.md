<h2 id="1"><strong>Windows Update 1803 Download:</strong></h2>
Every month, Microsoft will release updates for its Windows 10 product. The updates will enhance the security and features of the Windows OS. If you want to secure your system from malware attack, it is important that you have to perform an update process regularly.

If there are any bugs or vulnerabilities, the updates will be released randomly. A cumulative update means that it includes Window's specific version updates. If you install this update, your system will be up to date even if you don't have older updates.

You can install the Microsoft updates via Windows Update. You just have to go to Settings -&gt; Update &amp; Security -&gt; Check for updates so you can download and install the updates to your system.

But there are times when you have to download and install the updates manually. So, in this post, you will learn how to download any cumulative update for Windows 10. You can check out the list of all versions and choose the one as per your Windows version.

<h2 id="4">Steps to Install the Downloaded Cumulative Update:</h2>
<ol>
 	<li>Click on the <strong>Windows key</strong> to open the <strong>Start</strong> menu.</li>
 	<li>In the search box, you have to type <strong>‘cmd’</strong>. Right-click on the result and choose <strong>Run as administrator</strong>. (If prompted, you have to press <strong>‘Yes’</strong> in the User Account Control window.)

 	<li>You have to run the below command hit <strong>Enter</strong>. <code>wusa C:\Username\Downloads\windows10.0-kb4565489-x64_c79c926453b85267fb5cc5f969f172f6250b086c.msu</code></li>
 	<li>You can change the file path as per your system needs.</li>
 	<li>Add <strong>/silent</strong> switch at the end of the command to update silently.</li>
